Windows 安装部署 TANGO-Control

Author: Weiyunyi Date: 八月 9, 2025  15:46:19

1. 下载安装jdk

可参考b站教程：https://www.bilibili.com/video/BV1HG411E7Ue/?spm_id_from=333.1391.0.0&vd_source=2cbbfb29eb104eaddf4e845f1d91b08d

2. 安装配置mysql数据库

2.1 双击mysql-5.5.60-winx64.msi

2.2 安装过程

  1. 选择next

  2. 选择I accept……

  3. 选择Custom

  4. 点击change，并选择tango_root文件夹下的mysql文件夹，后点击next

  5. 点击Install

  6. 点击Next

  7. 点击Finish

  8. 点击Next

    ![image-20250809160934411](../images/Windows TANGO-Control image/image-20250809160934411.png)

  9. 选择Standard Configuation

  10. 取消勾选Modify Security Settings

  11. 点击Finish，即安装完成，接下来需要将mysql路径添加到环境变量中

2.3 将mysql添加到环境变量中

​ 1.选择 “设置 -> 系统 -> 系统信息 -> 高级系统设置 -> 高级”并点击 “环境变量”

​ 2.依次点击Path、编辑

![image-20250809161916856](../images/Windows TANGO-Control image/image-20250809161916856.png)

  1. 找到mysql安装路径下的bin文件夹，点击新建将路径进行添加，再依次点击确定

    ![image-20250809162436912](../images/Windows TANGO-Control image/image-20250809162436912.png)

3.安装TANGO

### 	3.1 双击TangoSetup-8.1.2b_win64.exe

​ ![image-20250809162840436](../images/Windows TANGO-Control image/image-20250809162840436.png)

### 	3.2安装过程

  1. 点击Next

    ![image-20250809162932247](../images/Windows TANGO-Control image/image-20250809162932247.png)

  2. 安装到tango_root文件夹下的tango文件夹中

    ![image-20250809162955762](../images/Windows TANGO-Control image/image-20250809162955762.png)

    ![image-20250809163009160](../images/Windows TANGO-Control image/image-20250809163009160.png)

  3. 点击Install

    ![image-20250809163040460](../images/Windows TANGO-Control image/image-20250809163040460.png)

  4. 两个全勾选上后，点击Finish

    ![image-20250809163058414](../images/Windows TANGO-Control image/image-20250809163058414.png)

  5. 点击Create

    ![image-20250809163114814](../images/Windows TANGO-Control image/image-20250809163114814.png)

3.3 添加系统变量

  1. 选择 “设置 -> 系统 -> 系统信息 -> 高级系统设置 -> 高级”并点击 “环境变量”

    ![image-20250809161631893](../images/Windows TANGO-Control image/image-20250809161631893-175473912871328.png)

  2. 点击新建，依次输入环境变量名、环境变量值，设置完成后如下图

    ![image-20250809163451340](../images/Windows TANGO-Control image/image-20250809163451340.png)

    ![image-20250809163457637](../images/Windows TANGO-Control image/image-20250809163457637.png)

3.4 添加路径

  1. 选择 “设置 -> 系统 -> 系统信息 -> 高级系统设置 -> 高级”并点击 “环境变量”

  2. 再依次点击Path、编辑，新建并输入“%TANGO_ROOT%\bin”,后依次点击确定完成设置

    ![image-20250809163834266](../images/Windows TANGO-Control image/image-20250809163834266.png)

3.5 继续配置

  1. 以管理员身份运行命令提示符，进入tango安装目录下的db文件夹，输入“dir” 回车，再输入“create_db.bat” 再回车，出现“Enter password:”后可直接回车

    ![image-20250809164231486](../images/Windows TANGO-Control image/image-20250809164231486.png)

  2. 进入到tango安装目录下的“bin”文件夹，再输入“dir”并回车

    ![image-20250809164511570](../images/Windows TANGO-Control image/image-20250809164511570.png)

  3. 输入“create_db.bat” 再回车，出现“Ready to accept request”即配置完成

    ![image-20250809164614006](../images/Windows TANGO-Control image/image-20250809164614006.png)

4. 安装完成后打开效果
